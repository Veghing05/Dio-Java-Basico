Variaveis, Nomeclatura

Uma variavel é sempre no singular, exceto quando se referir a
um array ou coleção.

defina um idioma único para as suas variáveis. Se você for 
declarar variaveis em inglês, defina todas em inglês.

Não Recomendado

double salMedio = 1500.23
String emails = "aluno@escola.com"
String myName = "Joseph"

Recomendado

double salarioMedio=1500.23
String email ="aluno@escola.com"
String [] emails = {"aluno@escola.com", professor@escola.com}
String meuNome = "Joseph
